# 데이터 타입

변수: 변할 수 있는 무언가
-> 변경 가능한 데이터가 담길 수 있는 공간 그릇

식별자: 변수명

## 자바스크립트 메모리 공간

변수 영역: 데이터가 담긴 주솟값이 담긴다
데이터 영역: 데이터가 담긴다.

## 깊은 복사 하는 법

- 재귀로 한다.
- JSON으로 바꾸었다가 다시 객체로

```js
var copyObjectViaJson = function (target) {
  return JSON.parse(JSON.stringify(target));
};

//getter나 setter __proto__ 같이 JSON으로 바꿀 수 없는 값은 복사 할 수 없다
```

## undefined

1. 값을 대입하지 않은 변수, 데이터 영역의 메모리 주소를 지정하지 않은 식별자에 접근
2. 객체 내부에 존재하지 않는 프로퍼티에 접근
3. return 문이 없거나 호출되지 않는 함수의 실행 결과

undefined가 아닌 비어있는 경우 순환할 수 없다. 그냥 존재하지 않는 프로퍼티에 접근할 수 없는거다.

EMPTY는 그냥 변수 자체가 없다.
undefined는 변수의 값이 존재하지 않음을 의미

직접 할당하지 않도록 하자 -> 이 경우 값을 대입하지 않은 변수에 접근하고자 할 때 자바스크립트 엔진이 반환해주는 값이 된다. -> 없음을 의미하고 싶을 때는 null을 주자
